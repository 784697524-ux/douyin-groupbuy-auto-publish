# 公开推广文案规则

## 可以写

- 消费场景：周末、约会、亲子、朋友聚会、看电影、逛商场
- 券包权益：价格、可用门店、可用日期、兑换方式、注意事项
- 购买引导：先看清规则，合适再下单，去之前先囤
- 本地化信号：城市、商场、门店、楼层、影城

## 不要写

- 商品 ID
- 后台售卖张数
- 成交金额
- 内部排名
- 后台截图
- 夸大承诺
- 评论关键词、关注私信、点赞收藏等诱导互动

## 推荐结构

1. 第一张图给钩子：价格/场景/人群。
2. 第二张图讲权益：几人用、包含什么、适合什么时候用。
3. 第三张图讲地点：商场/楼层/兑换点。
4. 第四张图讲规则：日期、退款、限制。
5. 第五张图做下单引导：去之前先看，合适就先囤。
